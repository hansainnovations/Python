### Tiling the Floor

# Instruction

You are going to tile your floor.**_`One`_** box of tile is having **_`5`_** pieces of tile and each is of **_`2 x 2 = 4 sqft`_**
You have to input **_`Length`_** and **_`breadth`_** of floor to calculate how many boxes of tiles you need to buy.

Number of Boxes = (Floor length **_`X`_** Floor breadth) / Coverage per box

e.g. Length=11 and Breadth = 13, Coverage = 20

number of boxes = (11 x 13) / 20 = 143/20 = 7.15

As you can not buy .15 box of tile **_`result should be rounded up to 8`_** boxes

**Hint** Function name and parameter must match the line code **_`13`_**

# Example Input

f_length=11

f_breadth=13

# Example Output

You need to buy **_`8`_** boxes of tiles
